The textbook makes heavy use of Sage cells (there are 96 vignettes with Sage cells, a list of which can be found here [Sage](https://books.aimath.org/fcla/appendix-SL.html)). These are intended to help students learn to use Sage to do linear algebra and to learn how to do linear algebra by using computational examples. There is a separate GitHub repository, (Rbeezer/SLA) that contains 17 Sage worksheets that instructors can use in class (in several output formats, including Jupyter, PDF, etc.).
